what is this fiddle?

wikiracer is a funny little game created by Marian Tietz (nemo) and Kristof Kipp (krizz).

the ultimate goal is to reach the target-wikipedia-page before your opponent/s do. later on there will be statistics about games, shortest and longest routes and whatever we can think of.

this thing is still in a very very very very early pre-alpha-stage so there might (and i bet there will be more than we expect) be some bugs. huehue.

btw.: you can find the exact code on github, isn't that awesome?
